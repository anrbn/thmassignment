# Running Locally

## 1. Run the THM Mock Webpage
```
cd thmassignment
npm install
npm start
```

## 2. Run the Mock APIs (AbuseDB & LLM)

```
cd thmassignment/labs/mock
python3 mockabusedb.py
python3 mockllm.py
```
